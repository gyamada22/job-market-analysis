# SQL Queries

Arquivo: `queries.sql`

Contém queries para análise das vagas:

1. Volume de vagas por senioridade
2. Top 3 skills obrigatórias por senioridade
3. Top 3 skills diferenciais por senioridade
4. Distribuição de vagas por estado e percentual
5. Distribuição por modalidade (Remoto, Híbrido, Presencial)
6. Top 5 setores por quantidade de vagas
7. Top 5 skills obrigatórias e diferenciais por setor (para os top 5 setores)

### Como usar

- Execute no SQL Server Management Studio (SSMS)  
- Pode ser utilizado como base para dashboards Power BI
